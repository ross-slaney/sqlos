## Auth Flow Summary

1. **Discover**: `POST /api/v1/auth/discover` with `{ email }` → returns login mode (password, sso)
2. **Password**: `POST /api/v1/auth/login` with `{ email, password, organizationId? }`
3. **SSO**: `POST /api/v1/auth/sso/start` → redirect to IdP → `POST /api/v1/auth/sso/exchange` with code
4. **OIDC**: `POST /api/v1/auth/oidc/start` → redirect to provider → callback → `POST /api/v1/auth/oidc/complete` with handoff
5. **Refresh**: `POST /api/v1/auth/refresh` with `{ refreshToken, organizationId? }`
6. **Session**: `GET /api/v1/auth/session` with `Authorization: Bearer <token>`
