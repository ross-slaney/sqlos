**Dashboard path**: Auth Server → OIDC

Apple support is web-only in v1 and requires a public HTTPS callback.

## Setup

1. Create a Services ID in the Apple Developer portal.
2. Enable Sign in with Apple.
3. Create the Apple OIDC connection in SqlOS.
4. Copy the connection ID from the OIDC connections list.
5. Register a final HTTPS callback URI using:

```text
https://your-public-host/api/v1/auth/oidc/callback/{connectionId}
```

6. Paste these fields into SqlOS:
   - Services ID / client ID
   - Team ID
   - Key ID
   - private key PEM

## Notes

- Apple uses a backend-generated client secret JWT.
- Apple may only return the user's name on first consent.
