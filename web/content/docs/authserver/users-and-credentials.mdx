**What this guide covers**: Create user, set password, user types (local, SSO, OIDC).

**Dashboard path**: Auth Server → Users  
**URL**: `http://localhost:5062/sqlos/admin/auth/users`

---

## Users Page

![Users page](/docs/users-page.png)

| Field | Required | Description |
|-------|----------|-------------|
| **Display name** | Yes | User's display name |
| **Email** | Yes | Primary email |
| **Password** | No | If set, user can sign in with email + password |

Users without a password can still sign in via SSO or OIDC.

## User Types

| Type | How created | Password |
|------|-------------|----------|
| **Local** | Dashboard, optional password | Optional |
| **SSO-provisioned** | SAML login, auto provision on | No |
| **OIDC** | Google, Microsoft, Apple, or custom OIDC login | No |

## Code Example

```csharp
var user = await adminService.CreateUserAsync(new CreateUserRequest
{
    DisplayName = "Jane Doe",
    Email = "jane@acme.com",
    Password = "securePassword123"
});
```
