# Sessions and Tokens

**What this guide covers**: Session creation, access token structure, refresh token rotation, token lifetimes.

---

Every successful login creates a session with an access token and a refresh token. The session tracks the user, client, authentication method, and expiration.

## Session Model

| Field | Description |
|-------|-------------|
| `Id` | Session identifier (prefix `ses_`) |
| `UserId` | Authenticated user |
| `ClientApplicationId` | Client that initiated login |
| `AuthenticationMethod` | `password`, `sso`, `oidc` |
| `IdleExpiresAt` | Resets on each refresh |
| `AbsoluteExpiresAt` | Hard expiration, cannot be extended |
| `RevokedAt` | Set on logout or forced revocation |

## Token Response

```csharp
var tokens = await authService.CreateSessionTokensForUserAsync(
    user, client, organizationId, "password",
    userAgent, ipAddress, cancellationToken);

// tokens.AccessToken      — JWT, short-lived
// tokens.RefreshToken     — opaque, rotated on use
// tokens.SessionId        — session reference
// tokens.ClientId         — issuing client
// tokens.OrganizationId   — scoped org (nullable)
// tokens.AccessTokenExpiresAt
// tokens.RefreshTokenExpiresAt
```

## Access Token

JWT signed with the RSA key from `SqlOSCryptoService`. Contains claims for user ID, session ID, client ID, organization ID, and custom claims. Validated via JWKS at `/.well-known/jwks.json`.

## Refresh Token

Opaque token stored as a hash. Each use rotates the token: the old token is marked as consumed and a new one is issued. Reuse of a consumed token triggers family revocation (all tokens in the chain are revoked along with the session).

## Lifetimes

Configured via security settings (dashboard or `SqlOSSettingsService`):

| Setting | Default |
|---------|---------|
| Access token lifetime | Configured in `SqlOSAuthServerOptions.AccessTokenLifetime` |
| Refresh token lifetime | From security settings |
| Session idle timeout | From security settings |
| Session absolute lifetime | From security settings |
