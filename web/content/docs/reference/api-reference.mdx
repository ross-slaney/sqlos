## Example Auth Endpoints (`/api/v1/auth`)

The example API wraps SqlOS services. **File**: `examples/SqlOS.Example.Api/Endpoints/ExampleAuthEndpoints.cs`

| Method | Endpoint | Request | Description |
|--------|----------|---------|-------------|
| POST | `/discover` | `{ email }` | Home realm discovery. Returns `{ mode: "password" \| "sso", organization?, ... }` |
| GET | `/oidc/providers` | - | List enabled OIDC providers for login UI |
| POST | `/login` | `{ email, password, organizationId? }` | Password login. May return `requiresOrganizationSelection` + `organizations` |
| POST | `/select-organization` | `{ pendingAuthToken, organizationId }` | Select org after multi-org discovery |
| POST | `/sso/start` | `{ email }` | Start SAML SSO. Returns `{ authorizationUrl, organizationId, ... }` |
| POST | `/sso/exchange` | `{ code, state }` | Complete SAML PKCE exchange. Uses cookies for verifier. |
| POST | `/oidc/start` | `{ email, connectionId }` | Start OIDC auth. Returns `{ authorizationUrl, ... }` or SSO redirect info |
| GET/POST | `/oidc/callback/{connectionId}` | `?code=&state=` or form post | Backend callback. Redirects to frontend with `handoff` or `error` |
| POST | `/oidc/complete` | `{ handoff }` | Exchange handoff for tokens |
| POST | `/refresh` | `{ refreshToken, organizationId? }` | Refresh access token |
| POST | `/logout` | `{ refreshToken?, sessionId? }` | Logout, revoke session |
| GET | `/session` | `Authorization: Bearer <token>` | Validate token, return user/session/claims |

## Code Examples

### Discover + Login Flow

```csharp
// 1. Discover login mode
var discover = await httpClient.PostAsJsonAsync("/api/v1/auth/discover",
    new { email = "user@acme.com" });
var mode = await discover.Content.ReadFromJsonAsync<DiscoverResponse>();

// 2. Password login
if (mode.Mode == "password")
{
    var login = await httpClient.PostAsJsonAsync("/api/v1/auth/login",
        new { email = "user@acme.com", password = "secret" });
    var tokens = await login.Content.ReadFromJsonAsync<TokenResponse>();
}

// 3. Refresh
var refresh = await httpClient.PostAsJsonAsync("/api/v1/auth/refresh",
    new { refreshToken = tokens.RefreshToken });
```

### FGA Check Access

```csharp
var result = await httpClient.PostAsJsonAsync("/sqlos/admin/fga/api/access-test",
    new { subjectId = "user-1", resourceId = "chain-1", permissionKey = "CHAIN_VIEW" });
```
